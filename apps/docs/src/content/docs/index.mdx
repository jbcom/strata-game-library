## Quick Start

Install the core package and peer dependencies:

```bash
pnpm install @strata-game-library/core @react-three/fiber @react-three/drei three
```

Build a complete scene in minutes:

```tsx
import { Canvas } from '@react-three/fiber';
import {
  ProceduralSky,
  Water,
  GrassInstances,
  VolumetricFogMesh,
} from '@strata-game-library/core';

function Game() {
  return (
    <Canvas camera={{ position: [0, 10, 20] }}>
      <ProceduralSky sunPosition={[100, 50, 100]} />
      <Water size={200} depth={20} />
      <GrassInstances count={10000} spread={100} />
      <VolumetricFogMesh density={0.02} />
    </Canvas>
  );
}
```

  Strata is evolving from a toolkit into a **complete game framework**. The upcoming `createGame()` API will enable fully declarative game definitions. [Read the roadmap ‚Üí](/getting-started/architecture/)

## Architecture

Strata is built in layers ‚Äî each one building on the last, like geological strata:

<div class="layer-stack">
  <div class="layer layer-4">Layer 4 &mdash; Presets & Game Framework</div>
  <div class="layer layer-3">Layer 3 &mdash; React Three Fiber Components</div>
  <div class="layer layer-2">Layer 2 &mdash; Core Algorithms (SDF, Noise, Marching Cubes)</div>
  <div class="layer layer-1">Layer 1 &mdash; GLSL Shaders</div>
  <div class="layer layer-0">Layer 0 &mdash; TypeScript Types & Utilities</div>
</div>

Every layer is independently usable. Import just the shaders, or use the full framework ‚Äî it's your choice.

## The Vision

<div class="mission-block">
  <blockquote>
    "Games should be defined, not engineered. Strata's goal is to make building 3D worlds as natural as writing a document ‚Äî declarative, composable, and beautiful by default."
  </blockquote>
</div>

The upcoming `createGame()` API will enable fully declarative game definitions:

```tsx
import { createGame, StrataGame } from '@strata-game-library/core';

const game = createGame({
  world: {
    terrain: 'alpine',
    water: { type: 'ocean', depth: 50 },
    vegetation: { density: 0.8, season: 'summer' },
    sky: { timeOfDay: 14, weather: 'partly-cloudy' },
  },
  creatures: [otter, deer, eagle],
  modes: { exploration, racing, combat },
  state: 'rpg',
});

<StrataGame game={game} />
```
