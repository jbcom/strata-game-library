    Module strata/core

    @strata-game-library/core

    Pure TypeScript algorithms for procedural 3D graphics. Terrain, water, vegetation, ECS, pathfinding, physics, animation, and more.

    For React Three Fiber components, use @strata-game-library/r3f.
