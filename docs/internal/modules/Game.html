    Module Game

    Game orchestration systems for managing scenes, modes, transitions, and triggers.

    Provides the core game lifecycle infrastructure: scene management for loading/unloading game levels, mode management for gameplay states (exploration, combat, dialogue), transition effects between scenes/modes, and a trigger system for event-driven game logic.
