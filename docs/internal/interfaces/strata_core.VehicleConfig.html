    Interface VehicleConfig

    Vehicle physics configuration for arcade-style or realistic driving. Controls chassis, wheels, suspension, steering, and motor behavior.

    Example

    const racingCarConfig: VehicleConfig = {
      chassisMass: 1200,
      wheelRadius: 0.35,
      motorForce: 2500,
      maxSteerAngle: Math.PI / 6,
      driveWheels: 'rear',
      // ...other properties
    };
